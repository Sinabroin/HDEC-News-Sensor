# D7-AD-Y 임원 대시보드 비주얼 톤 조사 (CSS 변경 전 · 계획 전용)

작성 시점: D7-AD-W Phase 1C. **이번 Phase에서는 CSS 대규모 리스킨을 하지 않는다.**
실제 스타일 변경은 별도 승인 후 D7-AD-Y에서 진행한다.

## 현재 인상

- 배경·카드·강조색에 **하늘색(sky) 계열**이 많아 전체 톤이 가볍게 느껴진다.
- 임원 보고용 대시보드로는 **점잖고(sober) 신뢰감 있는** 톤이 더 적합하다.

## 목표 톤 (조사 방향)

| 영역 | 방향 |
|---|---|
| 배경 | warm gray / off-white — 파란기 축소 |
| 텍스트 | navy / slate 계열 — 본문 대비 강화 |
| 강조 | sky 단독 → navy+amber 포인트(과하지 않게) |
| 카드 | 경계(line) 정돈, 그림자 약화 |
| 배지/pill | executive 보고서 스타일 — 과한 pill 색상 자제 |

## 금지

- 너무 어두운 다크 모드 / 군사 작전실 느낌
- 가짜 데이터·차트로 “풍성해 보이게” 만드는 장식
- 기능 IA 변경 없이 색만 바꾸는 것은 OK, 레이아웃 대개편은 별도 승인
