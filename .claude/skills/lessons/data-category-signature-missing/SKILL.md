## 실수
D4 검수에서 고득점 mock(mock_005)이 final_score 3.29로 4.5 미달.

## 원인
rules.md §7 고득점 카테고리 정의는 "AI 데이터센터 + 전력 + EPC/원전/송배전"인데,
mock_005 작성 시 원전/송배전 계열 키워드를 누락 → 가점 규칙(AI 전력수요+원전/SMR/송배전 +0.7)이
구조적으로 도달 불가능해짐. 데이터가 자기 카테고리의 점수 시그니처를 충족하지 못했다.

## 재발 방지 규칙
- mock 기사를 만들 때 해당 카테고리 정의(rules.md §7)의 필수 키워드 집합을 기사별로 체크한다.
- 고득점 기사는 가점 규칙 표(rules.md §6)에서 어떤 가점이 발동하는지 명시적으로 1건 이상 확인한다.
- D2 검수에 "카테고리 시그니처 키워드 포함 여부" 확인을 포함한다 (개수/길이 검증만으로는 부족).
